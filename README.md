# Network Troubleshooting Toolkit

A simple Linux-based network diagnostic toolkit built using Bash scripting.
It helps diagnose network connectivity issues using common networking commands.

## Features

- Ping Test (Check connectivity)
- Traceroute (Trace network path)
- IP Configuration Viewer
- DNS Lookup
- Log File Generation

## Tools Used

- Bash Shell
- ping
- traceroute
- nslookup
- ip command

## Installation

Clone the repository

git clone https://github.com/yourusername/network-troubleshooting-toolkit.git

Go to project folder

cd network-troubleshooting-toolkit

Make script executable

chmod +x network_toolkit.sh

Run the toolkit

./network_toolkit.sh

## Project Structure

network_toolkit.sh → main menu script  
scripts/ → individual network commands  
logs/ → stored troubleshooting logs
